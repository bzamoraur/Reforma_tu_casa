# MCP usage log

This project does **not** configure any MCP server automatically. MCP servers may be added later **only after explicit user approval**, and every use must be logged here.

## Policy

- No MCP server receives secrets or private user data.
- No MCP server is used for legal or construction truth.
- MCP config is local unless explicitly intended for the team.
- See the master prompt §18 for the allowed-later list and permissions.
